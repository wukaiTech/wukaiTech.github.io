1.如何比较两个Integer类型的数据？

使用“==”比较的是两个数据的地址值；

使用equals（）比较的是数据的内容；

因此一般使用equals（）来比较Integer类型的数据。

2.集合中，set、list、map的区别

  • set是无序不重复的集合，常见实现类有HashSet、TreeSet。
  • list是有序可重复的集合，常见的实现类有ArrayList、LinkedList。
    • ArrayList：基于动态数组的数据结构
    • LinkedList：基于链表的数据结构
    • 对于增加和删除操作，LinkedList要优于ArrayList，因为每插入一条数据，ArrayList就要移动插入点和插入点以后的所有数据。
  • map是用于存储键值对的集合，常见有HashMap、TreeMap。

4.对AOP的理解及其应用场景

面向切面编程（AOP）的设计模式，在该模式中，程序被分为核心业务逻辑和横切关注点两个部分。

切面类可以封装横切关注点的逻辑，将其与核心业务逻辑分隔开。

使用场景有：

  • 日志记录
  • 事务管理
  • 缓存管理
  • 权限控制
  • 异常处理

5.SpringBoot核心注解

核心注解是@SpringBootApplication，它主要由@SpringBootConfiguration，@EnableAutoConfiguration和@ComponentScan这三个构成。

  • @SpringBootConfiguration

    @SpringBootConfiguration里面就只有一个@Configuration主要注解，也就是把该类变成一个配置类所以 @SpringBootConfiguration就相当于@Configuration。

  • @EnableAutoConfigration

​ 让 Spring Boot 根据添加的依赖自动配置你的 Spring 应用。

  • @ComponentScan

​ 启动组件扫描特性，用于扫描标有@Component、@Service、@Repository、@Controller等注解的类，并注册为Spring的Bean。

  • 接口的默认方法和静态方法
  • 函数式接口
  • 新的时间日期 API
